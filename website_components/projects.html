Projects

These are my past and current projects!

Interactive Chrome Extension

I was a member of a 3-person team for our interactive Chrome extension project done at the Oasis organization at Northeastern University. I was responsible for the coding. We developed a Chrome extension that allowed the screen background to change and sound to play. We used HTML, CSS, and JavaScript to display all the features and used Figma to draw detailed mockups of our Chrome extension.

Bostonian Mobile App

I was a member of a 4-person team for the HackBeanpot Hackathon at Northeastern University. We developed a mobile app that facilitated finding attractions in Boston based on personal interests. We used React Native to create features (such as textboxes, maps, buttons, checkboxes, range sliders, etc.) for each page and to connect the pages together. We used CSS to make the pages appealing. We also applied software development practices (Kanban and GitHub) throughout the project.

Serious Games with AI Game

I led a 5-person group final project for the Serious Games with AI program at the MIT Beaver Works Summer Institute. I added Python code to a game that modeled the spread and control of diseases in a city. I also improved reinforcement learning algorithms for this game. Additionally, I used data analysis to hypothesize the contrast between human and machine performance when playing the game, and applied software engineering practices (such as Kanban, Scrum, and GitHub) throughout the project.
